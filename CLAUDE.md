# Pure — Golf Swing Analyzer

## Brand

- **Name:** Pure
- **Tagline:** "Swing pure"

## Design System

### Colors

| Role | Name | Hex |
|------|------|-----|
| Primary base | Blue Charcoal | `#021B22` |
| Primary accent | Cream | `#F6F1E5` |
| Secondary accent 1 | Forest Green | `#2E5B3B` |
| Secondary accent 2 | Pastel Yellow | `#F4D76A` |
| Pop accent | Cardinal Red | `#C53A3A` |

### Color Usage

- **Blue Charcoal (#021B22):** Core background / dark surfaces
- **Cream (#F6F1E5):** Text on dark backgrounds, light surfaces
- **Forest Green (#2E5B3B):** Secondary UI elements, success states
- **Pastel Yellow (#F4D76A):** Highlights, callouts
- **Cardinal Red (#C53A3A):** Small hits only — buttons, badges, alerts
